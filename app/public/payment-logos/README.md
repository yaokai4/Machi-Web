Payment brand SVGs in this folder are downloaded from Iconify's `logos` icon set API
and used as brand marks for the supported Stripe payment methods.

Sources:
- https://api.iconify.design/logos:apple-pay.svg
- https://api.iconify.design/logos:google-pay.svg
- https://api.iconify.design/logos:visa.svg
- https://api.iconify.design/logos:mastercard.svg
- https://api.iconify.design/logos:amex.svg
